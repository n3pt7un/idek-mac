# idek

A peer-to-peer iPhone control surface for macOS.

This repository hosts the **public website, privacy policy, and signed Mac releases** for idek. The application source code lives elsewhere.

## Download

Get the latest signed and notarized DMG from the [releases page](https://github.com/n3pt7un/idek-mac/releases/latest).

## Website

- Landing page: <https://n3pt7un.github.io/idek-mac/>
- Privacy policy: <https://n3pt7un.github.io/idek-mac/privacy/>

## System requirements

- **Mac:** macOS 14 (Sonoma) or later
- **iPhone companion app:** iOS 17 or later
- Both devices on the same local network
